## What I do

- Connect to the Neovim instance via MCP.
- The user might make changes to the files you modified. In that case, do not revert their changes.
  - When the user removes the code you generated, do not add it back.
  - However, if you think the change is incorrect or making things worse, confirm with the user.

## When to use me

Use this when a user asks you to pair program with them in Neovim.
